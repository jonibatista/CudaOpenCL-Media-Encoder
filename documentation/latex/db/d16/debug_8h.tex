\subsection{Detailed Description}
Macros das funções de depuração Macros que serão usadas nos programas desenvolvidos ao longo dos exemplos. Estas macros podem receber um número variável de parâmetros através de uma string de formatação como no \char`\"{}printf\char`\"{}. Seguem-\/se alguns exemplos: 
\begin{DoxyCode}
 DEBUG("i = %d e f=.2f%", i, f);
 ERROR("%s", msg);
\end{DoxyCode}
